AI Assistant
I have pre/post anxiety scores for two groups. Which test should I use?
PsyStat AI
For a 2×2 mixed design (pre/post × group), use a Mixed ANOVA — it tests the interaction between time and group. First, I'd check normality (Shapiro-Wilk) and sphericity. If violated, consider a non-parametric alternative or robust mixed model.

▶ Go to Stats Studio → Mixed ANOVA
What effect size should I report?
PsyStat AI
Report partial η² for the interaction effect. In clinical contexts, also report Cohen's d for the pre-to-post change within each group, plus the NNT if you can define a clinically meaningful threshold.

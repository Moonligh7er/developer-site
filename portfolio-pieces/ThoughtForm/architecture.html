How the App is Built

Thought+Form is built as a monorepo — one repository containing multiple packages that work together. The frontend (what you see) is a React web app. The backend (where data lives) runs on Cloudflare's servers. Desktop and mobile versions wrap the same web app in native shells.

Infrastructure — Where Things Run
☁
Cloudflare Workers
Runs all the backend code — like a server, but you don't manage hardware
~$5/mo base

Think of this as the "brain" of the app. When you click a button, the frontend sends a request to Cloudflare Workers, which processes it and returns data. It's serverless — you don't rent a computer, you just pay for the requests you use.

Account neededYes — Cloudflare account SetupRun bash packages/cloud/setup.sh to provision everything Deploywrangler deploy from packages/cloud/ Dashboarddash.cloudflare.com
◆
Cloudflare D1 (Database)
Stores all your data — projects, cards, contacts, stories, everything
~$0.75/M reads

D1 is a SQLite database that lives on Cloudflare's network. Every project, card, contact, narrative, decision, and setting is stored here. It's fast because it runs close to wherever you are in the world.

Created bywrangler d1 create thought-form-db ConfigDatabase ID in wrangler.toml line 16 Migrations19 migration files in packages/cloud/migrations/ Tables40+ tables (users, projects, cards, posts, decisions, narratives, etc.)
▤
Cloudflare R2 (File Storage)
Stores backups and file attachments — like a cloud hard drive
~$0.015/GB

R2 stores larger files that don't fit in a database — full backups of your data, card attachments, uploaded assets. It's similar to Amazon S3 but cheaper because Cloudflare doesn't charge for downloading files.

Created bywrangler r2 bucket create thought-form-backups Bucket namethought-form-backups

Anthropic Claude API
Powers ALL AI features — writing, analysis, agents, content generation
~$2-6/user/mo

Claude is the AI that powers every intelligent feature in the app: writing assistance, content generation, decision simulation, pattern analysis, agent workflows, and more. Users can either bring their own API key (BYOK — they pay Anthropic directly) or use the managed mode ($19/mo Pro includes 2M tokens).

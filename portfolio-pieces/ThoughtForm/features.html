Core Workspace

8 features
▣ Dashboard
Core

Your home screen. See project KPIs, recent activity feed, and launch quick actions -- all at a glance. Everything you need to start your session in one place.

// Technical Details
Polls /api/dashboard/stats for aggregated metrics. Real-time updates via Server-Sent Events (SSE). Card counts, velocity, deadline proximity, and streak tracking surfaced in a single payload.
▦ Kanban Board
Core

Full drag-and-drop task board with cards, columns, checklists, comments, attachments, card relations, and time tracking. Set WIP limits per column. Multiple card types and priority levels.

// Technical Details
Powered by dnd-kit for drag-and-drop. Supports card types (task, bug, feature, note) and priorities (critical, high, medium, low). WIP limits enforced per column. Optimistic locking via version column prevents concurrent edit conflicts. Relations modeled as card_relations table with typed edges (blocks, relates_to, parent_of).
★ Inception Framework
Core

Guided project scaffolding that walks you through defining scope, goals, audience, and milestones. AI suggests next steps and fills in gaps you might miss.

// Technical Details
Multi-phase checklist system stored in D1. Each phase (Define, Plan, Build, Launch) has configurable checklist items. AI suggestions powered by Claude API with project context injection. Completion state persisted per-project.
📅 Calendar
Core

Schedule events, deadlines, and milestones. Import/export iCal files. Supports recurring events with full recurrence rules. Sync with Google Calendar.

// Technical Details
Full RFC 5545 (iCalendar) support. RRULE parsing for complex recurrence patterns (daily, weekly, monthly, yearly with intervals and exceptions). Google Calendar sync via OAuth. Events stored in D1 with timezone-aware timestamps.
📖 Session Journal
Core

Your work diary. Log what you did each session with mood and energy tracking, tags, and free-form notes. Search across all entries instantly.

// Technical Details
Full-text search via SQLite FTS5 with session_fts index. Mood/energy stored as integers (1-5). Tags stored as JSON array. Entries linked to project via foreign key. Supports date-range filtering and tag-based aggregation.
📄 Snippet Vault
Core

Store and organize code snippets, text blocks, and templates with syntax highlighting and tags. Instant full-text search across all snippets.

// Technical Details
FTS5-powered search via snippet_fts index. Syntax highlighting via language detection. Soft-delete support (tombstone pattern) for safe recovery. Tags stored as JSON array. Copy-to-clipboard integration.
⌨ Clipboard Commands
Core

Custom shortcut buttons that instantly copy text or generate AI content to your clipboard. Static text or AI-powered dynamic generation.

// Technical Details
Two modes: static (predefined text) or AI-generated via /api/ai/quick. Each command stores a name, mode, and content/prompt. Clipboard API integration for one-click copy. Commands are per-project and orderable.
👥 Contacts & CRM
Core

Contact management with a deal pipeline. Track contacts through stages from lead to won. Log activities, notes, and interactions. Full-text search across all contacts.

// Technical Details
FTS5 search across contact fields. Deal stages: lead → qualified → proposal → negotiation → won → lost. Activity logging via contact_activities table (call, email, meeting, note). Contacts linked to projects, searchable by company, role, or tag.

Creative Suite

5 features
📑 Narrative Studio
Creative

AI-assisted writing environment supporting 9 formats: novel, screenplay, short story, marketing campaign, blog series, product docs, pitch deck, whitepaper, and newsletter. Hierarchical sections, context boxes for plot/setting/themes/tone, character color-coding, guided questionnaires, and AI generation (draft, continue, revise).

// Technical Details
Tables: narrative_projects, narrative_sections (tree structure via parent_id), narrative_context (keyed boxes), narrative_character_config (color assignments), narrative_questionnaires. Full context assembly for Claude: sections + context boxes + character profiles concatenated as system prompt. AI modes: draft (generate from outline), continue (append), revise (rewrite selection).
🌎 Worlds
Creative

Build fictional universes or brand worlds with structured sections: description, rules, geography, history, culture. AI expands any section on demand. Generate a complete wiki from your world data.

// Technical Details
Stored in worlds table with JSON sections. Claude API generates section expansions using full world context. Wiki generation outputs structured HTML with cross-linked sections, exportable as a standalone page.
🎬 Creative Stories
Creative

Outline stories with acts, beats, and themes. AI generates full outlines from a premise, enhances synopses, and expands individual scene beats into detailed sequences.

// Technical Details
Stored in creative_stories with acts as a JSON array. Each act contains ordered beats with type, summary, and expanded content. Claude API powers three generation modes: outline generation (premise → full structure), synopsis enhancement, and scene beat expansion.
👤 Characters
Creative

Deep character profiles: appearance, personality, backstory, motivations, relationships, and voice notes. AI portrait generation and voice design for bringing characters to life.

// Technical Details
Stored in creative_characters. AI portrait generation via Stability AI (SDXL) or OpenAI (DALL-E). Voice synthesis via ElevenLabs API for character voice design. Each character links to projects and can be referenced in Narrative Studio context.
🎵 Sound Design
Creative

Manage audio assets organized by mood and category. Preview and play audio files directly in the browser. Tag and search your sound library.

// Technical Details
Audio files stored in R2 object storage. Mood-based categorization (tense, calm, energetic, melancholy, etc.). Browser-native Audio API for playback. Metadata (duration, format, tags) indexed for search.

Intelligence Layer

13 features
⚡ AI Quick Ask
Intelligence

Direct chat with Claude. Ask questions, brainstorm ideas, get feedback -- with full conversation history preserved. Your AI co-founder, always on call.

// Technical Details
Claude API proxied through Cloudflare Worker. Conversations stored in ai_conversations and ai_messages tables. FTS5 search across all message content. Supports system prompt customization and temperature control.
⚙ AI Agent Workflows
Intelligence

Build multi-step autonomous workflows. Chain together AI prompts, data queries, entity writes, conditionals, and notifications. Trigger manually, on schedule, or via webhook.

// Technical Details
Step types: ai_prompt, query_data, write_entity, conditional, notify. Trigger modes: manual, scheduled (cron), webhook. Execution tracked in agents and agent_runs tables. Step executor processes sequentially with context passing between steps.
▶ AI Buttons
Intelligence

Create custom AI commands with tailored system and user prompts. Output to a panel, create a card, or save as a snippet. Set cost caps to control usage.

// Technical Details
Stored in ai_buttons with fields: name, system_prompt, user_prompt, output_mode (panel | card | snippet), cost_cap. Each invocation runs through the Claude API proxy with the configured prompts. Cost tracking per invocation against the cap.
🔬 Name Research
Intelligence

AI-powered naming for products, companies, or features. Checks domain availability and trademark conflicts. Scores each candidate on memorability, relevance, and availability.

// Technical Details
Stored in name_research with FTS5 search. AI generates candidates via Claude with scoring criteria. Domain checking via DNS lookup. Trademark search integration. Each name scored across multiple dimensions with an aggregate score.
⚖ Decisions Log & Impact Ledger
Intelligence

Track every important decision with options, pros/cons, and outcomes. The Impact Ledger traces decisions forward to completed cards, showing the real-world effect of each choice.

// Technical Details
Stored in decisions table with JSON options array (each with pros/cons). entity_links table connects decisions to cards, enabling impact tracing. Outcome tracking allows marking decisions as validated or revisited.
🧬 Project DNA
Intelligence

AI-generated personality profile for each project. Health scoring across 5 dimensions reveals blind spots and strengths. Understand your project at a deeper level.

// Technical Details
Health scores cached in project_health_cache across 5 dimensions: momentum, scope clarity, technical debt, team energy, and market fit. Claude analyzes project data (cards, journal entries, decisions) to generate scores and narrative insights. Refreshed on demand or periodically.
⏰ Temporal Patterns
Intelligence

Detects your productivity cycles, burnout risk, and velocity changes over time. Auto-respond triggers alert you when patterns shift. Know when to push and when to rest.

// Technical Details
Stored in temporal_patterns. Analyzes session journal timestamps, card completion rates, and energy levels to detect cycles. Burnout risk calculated from declining velocity + low energy streaks. Auto-respond via /api/patterns/auto-respond triggers configurable actions (notifications, journal prompts).
📈 Confidence Scoring
Intelligence

Per-card on-time probability predictions. See at a glance which tasks are likely to ship on schedule and which need attention.

// Technical Details
Calculated via /api/intelligence/confidence. Uses historical completion data mapped by card type to predict on-time probability. Factors: type accuracy mapping, time-in-column, dependency chain depth, and assignee velocity.
🔀 Context Switch Meter
Intelligence

Measures the cost of switching between projects. Recommends optimal batch sizes to minimize thrashing and maximize deep work.

// Technical Details
Analyzed via /api/intelligence/context-switching. Tracks project transitions from session journal data. Calculates switch cost (time lost per transition) and recommends optimal batch sizes based on historical patterns.
📚 Narrative Timeline
Intelligence

Your project's story told chronologically. Auto-generated events create a timeline you can browse like chapters. See the narrative arc of your work.

// Technical Details
Events stored in narrative_events. Auto-generated from significant actions (milestones hit, decisions made, features shipped). Weekly chapter grouping creates a readable narrative. Events can be manually annotated.
🔗 Relationship Graph
Intelligence

Visualize connections between all entities -- cards, decisions, contacts, stories, characters. See how everything in your project is interconnected.

// Technical Details
Built from card_relations and entity_links tables. Renders as a force-directed graph. Nodes represent entities (cards, decisions, contacts), edges represent typed relationships. Filterable by entity type and relationship type.
📷 Snapshots
Intelligence

Capture point-in-time project state. Compare any two snapshots to see exactly what changed -- cards added, moved, completed, or removed.

// Technical Details
Full project state serialized as JSON snapshot. Diff engine compares two snapshots and outputs structured change lists (added, removed, modified entities). Useful for sprint reviews and retrospectives.
✅ Approval Workflows
Intelligence

Gate important actions behind approval steps. Approve, reject, modify, or rollback. Keeps a full audit trail of every decision.

// Technical Details
Stored in approvals table with states: pending, approved, rejected, rolled_back. Each approval links to the triggering entity and action. Supports modification (approve with changes). Full audit trail with timestamps and reasons.

Empire Builder

11 features
💰 Revenue Intelligence
Empire

Track MRR, ARR, churn, and LTV across all your products. Pull data from Stripe, Gumroad, and Patreon. Portfolio-level aggregation shows your total empire health.

// Technical Details
Revenue events stored in revenue_events table. Multi-source ingestion via Stripe webhooks, Gumroad API, Patreon API. Metrics calculated: MRR, ARR, churn rate, LTV, net revenue retention. Portfolio aggregation sums across all products for empire-level dashboards.
📣 Brand Voices & Content Factory
Empire

Define brand profiles with voice DNA (tone, vocabulary, style). Turn one idea into 8 platform-specific content variants, all in your brand voice. One seed, infinite reach.

// Technical Details
Brand profiles in brands table with voice DNA fields (tone, vocabulary, banned words, style rules). Content seeds in content_seeds. Platform formatting rules define constraints per platform (char limits, hashtag rules, media requirements). Claude generates variants using brand voice as system prompt.
📰 Content Hub
Empire

Manage posts through their full lifecycle: draft, scheduled, published. Use templates for consistent formatting. Maintain an asset library for images and media.

// Technical Details
Posts in posts table with status enum (draft, scheduled, published, archived). Templates in content_templates with variable interpolation. Assets stored in R2 via assets table. FTS5 search across post content. Scheduled publishing via Cloudflare Worker cron triggers.
🌐 Social Dashboard
Empire

Multi-platform account management for Twitter, LinkedIn, Facebook, Instagram, Product Hunt, and Patreon. Unified analytics across all platforms.

// Technical Details
Accounts in social_accounts with OAuth tokens per platform. Analytics in social_analytics tracking followers, engagement rate, impressions, and clicks. Platform-specific API integrations. Unified dashboard aggregates metrics across all connected accounts.
▼ Audience Funnels
Empire

Build conversion funnels with stage tracking and source attribution. See where your audience comes from and where they drop off.

// Technical Details
Funnels defined in funnels table with ordered stages. Events tracked in funnel_events with timestamp, source, and metadata. Conversion rates calculated between stages. Source attribution tracks which channels drive the most conversions.
🔨 Product Forge
Empire

Idea-to-launch pipeline. Score ideas across 5 dimensions, run validation checklists, and track progress from concept to shipped product.

// Technical Details
Ideas in product_ideas with 5-dimension scoring matrix (market size, feasibility, uniqueness, revenue potential, passion). Validation checklists track customer interviews, prototype feedback, pricing tests. Pipeline stages from idea through validation, build, beta, and launch.
📡 Competitive Radar
Empire

Track competitor profiles with feature comparison matrices. Spot gaps in the market and identify where you have an advantage.

// Technical Details
Competitors stored in competitors table with structured profiles. Feature matrix via /api/competitors/matrix generates comparison grids. Gap detection highlights features competitors have that you lack (and vice versa). Notes and update history tracked per competitor.
🔄 Portfolio Synergy
Empire

Cross-product analysis that detects shared infrastructure, cross-sell opportunities, and resource imbalances across your entire portfolio.

// Technical Details
Synergy detection algorithms analyze: shared tech stacks (infra reuse), overlapping audiences (cross-sell), and resource allocation imbalances (time/money skew). Generates recommendations for consolidation and optimization across all projects.
🎨 Portfolio Site Builder
Empire

Visual site builder to showcase all your products. Pick a theme, preview live, and deploy. Your public-facing portfolio, built from your project data.

// Technical Details
Config-driven HTML generation from project metadata. Theme system with customizable colors, layouts, and sections. Live preview in-app. One-click deploy to Cloudflare Pages or custom domain. Pulls product descriptions, screenshots, and links from project data.
🕐 Ritual Engine
Empire

Scheduled reviews at every cadence: daily standup, weekly review, monthly retro, quarterly strategy. AI generates rich reports from your actual data.

// Technical Details
Rituals defined in rituals table with cadence (daily, weekly, monthly, quarterly). Execution tracked in ritual_runs. Data-enriched prompts pull cards completed, journal entries, decisions made, and revenue changes for the period. Claude generates actionable review reports.
⚡ Automation Rules
Empire

Event-driven if-this-then-that rules. When something happens in your project, automatically trigger actions -- move cards, send notifications, create entries.

// Technical Details
Pipelines in automation_pipelines with trigger/action pairs. Triggers: entity events (card moved, created, completed), time-based, threshold-based. Actions: move card, create card, send notification, update field, run AI prompt. Supports chaining multiple actions per trigger.

Developer Tools

5 features
🐙 GitHub Integration
Developer

Browse repos, commits, pull requests, and issues without leaving Thought+Form. Your code and your project management in one place.

// Technical Details
GitHub REST API integration authenticated via Personal Access Token (PAT). Fetches repos, commit history, PR status, and issues. Links GitHub issues to Kanban cards for traceability. Webhook support for real-time PR/issue updates.
🚀 DevOps Monitor
Developer

Track deployments across GitHub Actions, Cloudflare, and Render. See deploy status at a glance. One-click rollback when things go wrong.

// Technical Details
Deploy records stored in deploy_records with status (pending, success, failed, rolled_back), provider, commit SHA, and timestamps. Integrates with GitHub Actions API, Cloudflare API, and Render API. Rollback triggers redeployment of the previous successful build.
▒ Terminal
Developer

Full embedded terminal right in the app. Run commands, scripts, and tools without switching windows. Desktop only.

// Technical Details
Powered by xterm.js frontend + node-pty backend via Fastify sidecar WebSocket. Full PTY emulation with resize support. Persistent sessions across tab switches. Desktop-only feature (requires local sidecar process).
📁 AI Filesystem Access
Developer

Let AI read and write files in authorized project folders. Sensitive files are automatically blocked. Safe, scoped filesystem access for AI workflows.

// Technical Details
Sidecar validates all paths via realpath to prevent directory traversal. Authorized folders configured per-project. Sensitive file patterns blocked (.env, credentials, keys). 1MB file size cap per read/write operation. Full audit logging of all AI filesystem operations.
🤖 Claude Code Delegation
Developer

Delegate coding tasks to Claude Code CLI directly from the app. Read-only by default for safety. Let AI implement features, fix bugs, and write tests.

// Technical Details
Invokes claude -p in headless mode via sidecar. Read-only filesystem access by default (user must explicitly grant write). 5-minute execution timeout. Output streamed back to the app in real-time. Supports passing project context and specific task instructions.

System

8 features
🔍 Global Search
System

Search across everything with Ctrl+Shift+F. Cards, journal entries, snippets, contacts, decisions, AI conversations -- all searchable from one bar.

// Technical Details
Unions multiple FTS5 tables (cards, sessions, snippets, contacts, AI messages, decisions) into a single search result set. Ranked by relevance with entity type indicators. Keyboard shortcut: Ctrl+Shift+F.
⚡ Quick Switcher
System

Command palette with fuzzy matching. Hit Ctrl+K to jump to any project, page, or action instantly. Navigate at the speed of thought.

// Technical Details
Zustand-powered state with keyboard navigation (arrow keys, enter, escape). Fuzzy matching algorithm scores results by character proximity. Indexes projects, pages, recent items, and commands. Keyboard shortcut: Ctrl+K.
🔌 Offline Sync
System

Keep working when you lose connection. Changes queue locally and sync automatically when you reconnect. No data lost, no interruptions.

// Technical Details
Offline mutations queued in IndexedDB with timestamps. Service worker intercepts failed network requests and queues them. On reconnect, queued mutations replay in order against the cloud API. Conflict resolution uses last-write-wins with server timestamp authority.
💾 Backup & Export
System

Full data export and import. Download everything as JSON. Restore from backup at any time. Your data is always yours.

// Technical Details
Export serializes all project data to JSON. Backups stored in R2 object storage with versioning. Import validates schema integrity before restoring. Supports selective restore (specific projects or entity types).
📡 RSS Autoposter
System

Subscribe to RSS/Atom feeds and automatically create draft content posts from new items. Stay on top of industry news and turn it into content.

// Technical Details
RSS/Atom feed parser with scheduled polling. SSRF protection validates feed URLs against private IP ranges. New items create draft posts in Content Hub with source attribution. Configurable filters (keyword match, author, category).
💳 Billing
System

Free tier with generous limits. Pro at $19/month with 14-day free trial. Stripe-powered payments. Bring your own API key (BYOK) for AI features on any tier.

// Technical Details
Stripe integration with webhook handlers for subscription lifecycle (created, updated, cancelled, payment_failed). Pro gate middleware checks subscription status on protected endpoints. BYOK mode allows users to provide their own Claude API key, bypassing metered AI usage. 14-day trial with full Pro access.
🎨 Themes
System

Three visual themes: CRT (phosphor green scanlines), Dark (clean and modern), and Void (minimal). Accessibility mode strips all CRT effects for reduced motion.

// Technical Details
CSS custom properties swapped per theme. CRT theme adds scanline overlay, glow effects, and phosphor green palette. Dark theme uses neutral grays. Void theme is ultra-minimal. Accessibility mode respects prefers-reduced-motion and disables all animations, scanlines, and glow effects.
📱 Progressive Web App
System

Install Thought+Form as a native-feeling app on any device. Service worker enables offline caching. Same React codebase, reduced feature set (no terminal/git).

// Technical Details
Service worker with cache-first strategy for static assets and network-first for API calls. Web App Manifest enables install prompt on supported browsers. Shares the React codebase with the desktop app but excludes sidecar-dependent features (terminal, git, filesystem, Claude Code delegation).

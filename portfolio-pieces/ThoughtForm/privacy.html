Privacy Policy

Last updated: March 28, 2026
The short version: We collect the minimum data needed to run the Service. We don't collect your email, name, or any personal identifiers. We don't track you. We don't sell data. You own everything you create. You can export it all anytime.

1. Information We Collect

What we store

  • Passphrase hash (never plaintext)
  • Recovery code hash
  • Your project data (cards, stories, contacts, etc.)
  • AI conversation history
  • Encrypted API keys (if BYOK mode)
  • Stripe customer/subscription IDs (if Pro)
  • Refresh token hashes (for session management)

What we DON'T collect

  • Email address
  • Real name
  • Phone number
  • IP address logs
  • Analytics or telemetry
  • Tracking pixels or cookies
  • Third-party advertising data
  • Device fingerprints
  • Location data

Thought+Form uses passphrase-based authentication with no email or username. We cannot identify you as a person — only authenticate your passphrase.

2. How Data Is Stored

  • Database: Cloudflare D1 (SQLite) — encrypted at rest by Cloudflare's infrastructure.
  • File storage: Cloudflare R2 — for backups and attachments.
  • Computation: Cloudflare Workers — serverless, no persistent servers.
  • Desktop app: Data syncs from cloud. Local sidecar handles terminal, git, and filesystem access — this data stays on your machine.

All data is stored in Cloudflare's global network with encryption at rest. We do not maintain our own servers.

3. API Key Handling

If you use Bring Your Own Key (BYOK) mode, your API keys (Claude, GitHub, social platforms, etc.) are encrypted with AES-256-GCM using a key derived from your passphrase. We cannot read your API keys — decryption requires your passphrase, which we never store in plaintext.

In Managed AI mode, we proxy requests through our own API key. Your content is sent to Anthropic's Claude API for processing.

4. AI Data Processing

When you use AI features (Quick Ask, Narrative Studio, Agent Workflows, content generation, etc.):

  • Your prompts and context are sent to Anthropic's Claude API for processing.
  • Anthropic's data handling is governed by their privacy and usage policies.
  • AI conversation history is stored in our database for your reference.
  • We do not use your AI conversations to train models or improve our service.
  • Token usage is tracked for billing purposes (counts only, not content).

5. Third-Party Services

Thought+Form integrates with the following third-party services. When you connect these services, data flows between T+F and the provider:

Service Purpose Data shared
Stripe Payment processing Customer ID, subscription status
Anthropic (Claude) AI features Prompts, context, conversation content
GitHub Code integration OAuth token, repo/commit/PR data
Social platforms Content publishing OAuth tokens, post content
Google Calendar Calendar sync OAuth token, event data
Stability AI / OpenAI Image generation Image prompts (via your API key)
ElevenLabs Voice synthesis Voice prompts (via your API key)

All third-party integrations are opt-in. We do not share your data with any service you haven't explicitly connected.

6. Cookies and Tracking

We do not use tracking cookies, analytics, or telemetry.
  • Authentication cookies: HttpOnly, Secure, SameSite=Strict cookies are used for session management in the PWA. These are functional cookies required for the service to work — not tracking cookies.
  • No analytics: We do not use Google Analytics, Mixpanel, Segment, or any analytics service.
  • No tracking pixels: No Facebook Pixel, LinkedIn Insight Tag, or similar.
  • Advertising (free tier only): Free-tier users may see ads served by Google AdSense. Google uses cookies to serve ads based on your prior visits to this or other websites. You may opt out of personalized advertising by visiting Google's Ads Settings. For more information about how Google uses data, see How Google uses information from sites that use our services. By default, Thought+Form requests non-personalized ads to minimize cookie usage. Pro users see no ads.

7. Data Retention

  • Your data is retained for as long as your account exists.
  • Deleted items (cards, contacts, snippets) use soft delete — they can be recovered. Hard deletion removes data permanently.
  • AI conversation history is retained until you delete it.
  • Backups stored in R2 are retained until you delete them.
  • If you stop using the Service, your data remains until you explicitly request deletion or your account is terminated.

8. Data Export

You can export all your data at any time via the Backup & Export feature in Settings. Exports include all projects, cards, contacts, stories, characters, AI history, and configuration. The export is a structured JSON file that you own completely.

9. Security Measures

  • Passphrase hashing: PBKDF2 with salt (Web Crypto API)
  • API key encryption: AES-256-GCM with passphrase-derived key
  • Transport: HTTPS enforced (HSTS with 1-year max-age)
  • Tokens: JWT access tokens (1hr expiry) + hashed refresh tokens (30-day expiry)
  • Rate limiting: Per-IP sliding window — 10/min for auth, 120/min general
  • CORS: Origin whitelist with credentials support
  • Security headers: CSP, X-Frame-Options: DENY, X-Content-Type-Options, Referrer-Policy
  • Input validation: Zod schemas on all API inputs
  • SQL injection: All queries use parameterized binding
  • Desktop filesystem: Authorized-folder whitelist with symlink traversal prevention

10. Children's Privacy

Thought+Form is not intended for use by individuals under the age of 13. We do not knowingly collect data from children. If you believe a child has used the Service, please contact us and we will take appropriate action.

11. International Data

Thought+Form uses Cloudflare's global network, which means your data may be processed in multiple countries. Cloudflare's infrastructure complies with applicable data protection regulations. By using the Service, you consent to the transfer and processing of data across borders as necessary to provide the Service.

12. Your Rights

Depending on your jurisdiction, you may have the right to:

  • Access your data (available via the Export feature)
  • Delete your data (available via account settings)
  • Port your data (JSON export)
  • Object to processing (you can stop using the Service at any time)

Since we don't collect personal identifiers (no email, no name), most data subject requests are handled through the app's built-in features rather than a formal request process.

13. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected by updating the "Last updated" date. Continued use of the Service after changes constitutes acceptance.

14. Contact

For privacy-related questions, please reach out through the project's GitHub repository or the contact methods provided within the application.

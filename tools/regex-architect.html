Regex Cheat Sheet

Character Classes
. Any character (except newline)
\d Digit [0-9]
\D Not a digit
\w Word char [a-zA-Z0-9_]
\W Not a word char
\s Whitespace
\S Not whitespace
Quantifiers
* 0 or more
+ 1 or more
? 0 or 1 (optional)
{n} Exactly n times
{n,} n or more times
{n,m} Between n and m times
*? Lazy (non-greedy) match
Anchors & Boundaries
^ Start of string/line
$ End of string/line
\b Word boundary
\B Not a word boundary
(?=...) Positive lookahead
(?!...) Negative lookahead
(?<=...) Positive lookbehind
Groups & Alternation
(...) Capturing group
(?:...) Non-capturing group
(?<name>...) Named group
a|b Alternation (a or b)
[abc] Character set
[^abc] Negated set
[a-z] Range
Common Escape Sequences
\n Newline
\t Tab
\\ Literal backslash
\. Literal period
\* Literal asterisk
\+ Literal plus
\? Literal question mark
Flags
g Global (find all matches)
i Case-insensitive
m Multiline (^ $ per line)
s Dotall (. matches \n)
u Unicode support
y Sticky (from lastIndex)
